### Parser

Script to parse transcript from rada.gov.ua

Example:

```
python url_parser.py https://www.rada.gov.ua/meeting/stenogr/show/8124.html ./8124.json
```

## Aligner

Usage:

```
from aligner.whisperx-aligner import whisperx_alignment

input_audio_file = "sample.wav"
input_text_file = "sample.txt"
output_file = "sample.json"

whisperx_alignment(input_audio_file, input_text_file, device="cuda", output_file=output_file)
```
